Двоичные сортированные деревья

Задача - составить двоичное сортированное дерево библиографических источников 
(по автору, по названию, по изданию), используя набор файлов, 
использовать двоичное дерево для создания и одновременной сортировки.

Программа должна считывать набор текстовых файлов в формает bib, выделяя из них
записи об отдельных книгах.

Вход программы - список файлов, которые необходимо обработать.
Выход программы - ФАЙЛ, в котором записи о книгах упорядочены в алфавитном порядке,
по авторам, для одного автора - по названиям книг.

Использовать структуру вида
(самостоятельно создать структуру на основании структуры bib).
Реализовать функции поиска слова в дереве по первым 5 символам (использовать
strncmp(..) для английских слов  или ее аналог (написать самим!) для одной из  
русских кодировок , лучше для cp1251 для сравнения слов по первым символам).
Таблица кодировки в данном каталоге размещена.
(Для изменения кодировки использовать функции  из библиотек iconv)
