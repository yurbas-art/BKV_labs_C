
Задача - прочитать посимвольно текстовой файл, содержащий
слова на английском языке, числа и разделители разного типа.
Имя файла ввести с клавиатуры.
Проверить результат открытия файла.
Использовать функцию fgetc для считывания из файла. 

Подсчитать:
общее количество символов;
количество строк,  ( по количеству символов "новая строка" - '\n' );
количество цифр, ( используя макрос isdigit(c) );
количество знаков препинания, ( используя макрос ispunct(c) );
количество пробельных символов, ( используя макрос isspace(c) );
количество букв,
 ( используя макрос isalpha(c) );
 из них - количество строчных, ( используя макрос islower(c) );
количество прописных букв, ( используя макрос isupper(c) );
количество гласных,
 ( написать свою функцию );
количество согласных;
Вывести результат на стандартный вывод.

Придется написать функции, определяющие принадлежность
символа к гласным или согласным буквам. Для остального
использовать функции (макросы) из заголовочного файла
ctype.h вида is* (isdigit, isalpha, ispunct и т.п.).
