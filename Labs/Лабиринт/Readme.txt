  Прохождение лабиринта

Лабиринт прямоугольной форма размером MxN состоит из квадратных площадок, возможно
отделенных друг от друга стенами. Перейти на соседнюю
площадку можно только если между ними нет стены.
Карта лабиринта состоит из шестнадцатеричных цифр, по N
в каждой из M строк. Соответствие наличия стенок у площадки
шестнадцатеричному коду показано на рисунке codes.png,
пример простого лабиринта приведен на рисунке simple.png.
Написать программу, находящую самый короткий путь от 
верхней левой площадки до правой нижней. Вывести на экран
длину пути (количество шагов, которые нужно сделать до достижения цели).
В файл с именем output.txt вывести карту лабиринта, где
пройденные кратчайшим путем площадки будут содержать символ '*'.
Для тестирования использовать прилагаеме файлы.
